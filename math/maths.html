Differential Calculus formule

  • dx/dx = 1
  • d(constant)/dx = 0
  • dxn/dx = nxn-1
  • d(constant)xn = (const)nxn-1
  • d(m+-n)/dx = dm /dx+dn/dx
  • dsinx/dx = cosx
  • dcosx/dx = -sinx

Trignometry function Formule

  • cos(-x) = cosx
  • sin(-x) = -sinx
  • cos(x+y) = cosxcosy - sinxsiny
  • cos(x-y) = cosxcosy + sinxsiny
  • cos(π/2 - θ) = sinx
  • sin(x+y) = sinxcosy + cosxsiny
  • sin(x+y) = sinxcosy - cosxsiny
  • cos(π/2 + θ) = -sinx
  • sin(π/2 + θ) = cosx
  • cos(π + x) = -cosx
  • sin(π + x) = -sinx
  • tan(x+y) = (tanx + tany)/1-tanxtany
  • tan(x-y) = (tanx - tany)/1+tanxtany
  • cot(x+y) = (cotxcoty - 1)/coty+cotx
  • cot(x-y) = (cotxcoty + 1)/coty-cotx
