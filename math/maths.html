Differential Calculus formule

  • dx/dx = 1
  • d(constant)/dx = 0
  • dxn/dx = nxn-1
  • d(constant)xn = (const)nxn-1
  • d(m+-n)/dx = dm /dx+dn/dx
  • dsinx/dx = cosx
  • dcosx/dx = -sinx

Note : Its is used when changes are small

Intregal Calculus formule

  • dx/dx = 1
  • d(constant)/dx = 0
  • dxn/dx = nxn-1
  • d(constant)xn = (const)nxn-1
  • d(m+-n)/dx = dm /dx+dn/dx
  • dsinx/dx = cosx
  • dcosx/dx = -sinx

Note : when we want to add changes upto infinite times
